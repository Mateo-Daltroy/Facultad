

# DIRECCIONAMIENTO
> El routeo es el proceso por el cual se encuentran dos dispositivos conectados a internet a traves del protocolo IP
- El routeo en IP, el routeo se hace hop-by-hop, lo cual quiere decir que cada salto se hace en direccion al proximo salto mas cercano a la direccion deseada hasta llegar
- La unidad basica transportada por el protocolo se llama datagrama
