
# Clase 1

* Get:
  - Cuerpo vacio
  - Cacheable
  - Menos seguro para transferencia de datos

* Post:
  - Cuerpo con datos
  - No Cacheable
  - Mas seguro para datos mas sensibles

* Head
  - Get pero solo de headers

## Headers
> Los headers reunen comandos que son muy comunes en una comunicacion http y se les da un titulo par aestandarizar

## TLS
- SSL/TLS como medio de seguridad se vuelve obligatorio en HTTP 2.0
